# Aula 4 - RTOS, Tarefas e Prioridade

Projeto da aula demonstrando criação de tarefas com **FreeRTOS**.

## Simulação no Wokwi
https://wokwi.com/projects/305569599398609473

## Arquivos
- codigo.ino → código da simulação
- screenshot.png → print da execução no Wokwi

## Descrição
O código cria duas tarefas:

- **TaskHigh** → prioridade maior
- **TaskLow** → prioridade menor

Cada tarefa escreve mensagens no Serial Monitor usando FreeRTOS.
